Magoose game is a side scrolling game with the objective of consuming items on the screen while avoiding deadly cacti. The more you eat the more projectiles you earn larger you grow. Dodge or Die
Previous Next

Dodge or Die is game where the goal is to reach the top of the screen. The trick is to kill yourself using the white blocks, and on the next life you can climb on top of your previous body which has not gone away (this was originally a bug, but now a feature!). See README for instructions. Written in javascript using the p5 and p5.play libraries.

Fusor
Previous Next

The Farnsworth–Hirsch fusor here uses electrostatic containment to attain nuclear fusion in a relatively small machine. Purple glow comes from the plasma in the near vacuum chamber while collsions happen in the center. Built completely from scratch, this project allowed me to explore new areas circuitry and machining.

SketchPad
Previous Next

This web application is a simple sketchpad, where you can draw dots, have them point to the center as lines, or animate them. Animating it with too many dots can stress the computer, so it's best to limit the number of dots before animating. It's built with JavaScript and FireBase so anyone on the same page can draw on the same canvas.
